# sabokit-assets

Static assets hot-linked from sabokit installs. `application-icons/` holds
third-party app logos used as nominative identifiers on the SSO dashboard —
each one points to the actual product it depicts. Files are referenced from
pinned release tags, so renames break installs; add a new file alongside and
bump the tag instead of replacing in place.

Icons remain the property of their respective rights holders. No ownership
claimed.

If you hold rights to a logo here and want it removed, email
shey.alnasrawi@proton.me.
